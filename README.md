# Dina Khoma Jewelry

This is a Dina Khoma site. Includes:

- MDX and Markdown support
- Optimized for SEO (sitemap, robots, JSON-LD schema)
- RSS Feed
- Dynamic OG images
- Syntax highlighting
- Tailwind v4
- Vercel Speed Insights / Web Analytics
- Geist font

Then, run Next.js in development mode:

```bash
npm run dev
```
